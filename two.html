Installing MySQL

MySQL Installation

  1. Download the MySQL Installer from the official MySQL website.
  2. Run the installer and choose the setup type (Developer Default is recommended).
  3. Follow the installation wizard steps to install MySQL Server and other tools.
  4. Set a strong root password when prompted during configuration.
  5. Start the MySQL Server and complete the configuration process.

Setting Up MySQL Environment

  1. Ensure that the MySQL service is running after installation.
  2. Set up environment variables if needed:
    • Windows: Add the MySQL bin folder (e.g., C:\Program Files\MySQL\MySQL Server 8.0\bin) to the PATH variable.
    • Linux/macOS: Add export PATH=$PATH:/usr/local/mysql/bin to ~/.bashrc or ~/.zshrc, then run source ~/.bashrc.
  3. Verify installation by running mysql --version in your terminal or command prompt.
